MeatHITShop

Online meat store with modern design and an easy shopping experience.

Project Description

MeatHITShop is an e-commerce platform that allows users to browse, select, and purchase meat products online. The project is built using Next.js and follows the FSD (Feature-Sliced Design) architecture for clean and scalable code.

Features

Product catalog with images and descriptions

Add products to the cart

Checkout process

Filters and search for products

Responsive design for mobile and desktop

Technologies

Next.js – frontend framework

React – UI components

TypeScript – type safety

FSD (Feature-Sliced Design) – project architecture

CSS / Tailwind – styling

Folder Structure (FSD)
src/
app/ # global pages, Next.js routing
pages/ # Next.js pages (if using pages directory)
features/ # separate project features
entities/ # core entities (products, users)
shared/ # shared components, assets, utilities
widgets/ # complex UI components
Getting Started

# install dependencies

npm install

# run in development mode

npm run dev

# build the project for production

npm run build
npm start
